Introducción a JavaScript

JavaScript es un lenguaje de programación que permite implementar funciones complejas en páginas web.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, orientado a objetos, utilizado principalmente en el desarrollo web para añadir interactividad a las páginas.

Sintaxis básica

// Variables
let nombre = "Juan";
const edad = 30;
var esEstudiante = true;

// Función
function saludar(persona) {
  return "Hola, " + persona;
}

// Condicional
if (edad >= 18) {
  console.log("Mayor de edad");
} else {
  console.log("Menor de edad");
}

// Bucle
for (let i = 0; i < 5; i++) {
  console.log("Iteración: " + i);
}

Manipulación del DOM

El DOM (Document Object Model) permite a JavaScript interactuar con los elementos HTML de una página.

¿Qué es el DOM?

El DOM es una representación en forma de árbol de los elementos de un documento HTML. JavaScript puede modificar este árbol para cambiar el contenido, la estructura o el estilo de la página.

Código DOM

// Seleccionar elementos
const titulo = document.getElementById('titulo');
const parrafos = document.querySelectorAll('p');

// Modificar contenido
titulo.textContent = 'Nuevo título';

// Crear nuevo elemento
const nuevoParrafo = document.createElement('p');
nuevoParrafo.textContent = 'Este es un nuevo párrafo';
document.body.appendChild(nuevoParrafo);

// Cambiar estilos
titulo.style.color = '#f0db4f';
titulo.style.backgroundColor = '#333';

Resultado

Título original

Este es un párrafo existente.

Eventos

JavaScript puede responder a eventos del usuario como clicks, movimientos del ratón, pulsaciones de teclas, etc.

Tipos de eventos

Los eventos más comunes incluyen click, mouseover, mouseout, keydown, keyup, submit, change, load y muchos más. Puedes escuchar estos eventos con addEventListener().

Código de Eventos

const boton = document.getElementById('miBoton');
const contador = document.getElementById('contador');
let cuenta = 0;

// Evento click
boton.addEventListener('click', () => {
  cuenta++;
  contador.textContent = cuenta;
  boton.style.transform = 'scale(1.1)';
  setTimeout(() => {
    boton.style.transform = 'scale(1)';
  }, 200);
});

// Evento teclado
document.addEventListener('keydown', (event) => {
  if (event.key === 'Enter') {
    console.log('Enter presionado');
  }
});

Resultado

Contador: 0

Fetch API

La API Fetch proporciona una interfaz para recuperar recursos (incluyendo a través de la red).

Async/Await

async/await es una forma moderna de trabajar con promesas en JavaScript, haciendo que el código asíncrono sea más fácil de leer y escribir.

Código Fetch

async function obtenerUsuarios() {
  try {
    const respuesta = await fetch(
      'https://jsonplaceholder.typicode.com/users'
    );
    const usuarios = await respuesta.json();
    mostrarUsuarios(usuarios);
  } catch (error) {
    console.error('Error:', error);
  }
}

function mostrarUsuarios(usuarios) {
  const lista = document.getElementById('lista-usuarios');
  lista.innerHTML = usuarios
    .map(user => `<li>${user.name}</li>`)
    .join('');
}

document.getElementById('cargar-btn')
  .addEventListener('click', obtenerUsuarios);

    LocalStorage

    LocalStorage permite almacenar datos en el navegador del usuario de forma persistente.

    Almacenamiento web

    Además de localStorage, existe sessionStorage (datos se borran al cerrar la pestaña) y cookies (para pequeñas cantidades de datos).

    Código LocalStorage

    // Guardar dato (string o objeto)
    localStorage.setItem('clave', 'valor');
    localStorage.setItem('usuario', JSON.stringify({ nombre: 'Juan' }));

    // Mostrar todos los datos
    Object.entries(localStorage).forEach(([clave, valor]) => {
      console.log(`${clave}: ${valor}`);
    });

    // Exportar datos como JSON
    const datos = JSON.stringify(Object.fromEntries(Object.entries(localStorage)));
    const blob = new Blob([datos], { type: 'application/json' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'backup-localStorage.json';
    a.click();
    URL.revokeObjectURL(url);
