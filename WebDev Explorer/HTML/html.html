Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de una página mediante el uso de elementos y etiquetas.

¿Qué es HTML?

HTML no es un lenguaje de programación, sino un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usas para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

Estructura básica HTML

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi Página</title>
</head>
<body>
  <h1>Hola Mundo</h1>
  <p>Mi primera página web.</p>
</body>
</html>

Resultado

Hola Mundo

Mi primera página web.

Elementos Semánticos

HTML5 introdujo elementos semánticos que describen mejor su contenido tanto para el navegador como para el desarrollador.

Importancia de la semántica

Los elementos semánticos mejoran la accesibilidad, el SEO y hacen el código más legible. Ejemplos incluyen <header>, <nav>, <main>, <article>, <section>, <footer>, entre otros.

Código HTML Semántico

<header>
  <h1>Mi Sitio Web</h1>
  <nav>
    <a href="#">Inicio</a>
    <a href="#">Acerca de</a>
  </nav>
</header>
<main>
  <article>
    <h2>Artículo Principal</h2>
    <p>Contenido del artículo...</p>
  </article>
</main>
<footer>
  <p>Derechos reservados</p>
</footer>

Resultado

Mi Sitio Web

Inicio Acerca de

Artículo Principal

Contenido del artículo...

Derechos reservados

Formularios HTML

Los formularios permiten la interacción del usuario con la página web, recopilando datos y enviándolos a un servidor.

Elementos de formulario

Los formularios usan elementos como <input>, <textarea>, <select>, <button> y otros. Cada uno tiene atributos importantes como type, name, placeholder, required, etc.

Código de Formulario

<form action="/procesar" method="post">
  <div>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
  </div>
  <div>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
  </div>
  <div>
    <label for="mensaje">Mensaje:</label>
    <textarea id="mensaje" name="mensaje" rows="4" required></textarea>
  </div>
  <button type="submit">Enviar</button>
</form>

Resultado

Multimedia en HTML

HTML5 introdujo elementos nativos para incluir multimedia sin necesidad de plugins.

<video> y <audio>

Estos elementos permiten incrustar contenido multimedia directamente en tus páginas. Puedes controlar la reproducción con JavaScript y personalizar los controles.

Código Multimedia

<!-- Video -->
<video width="400" controls>
  <source src="video.mp4" type="video/mp4">
  Tu navegador no soporta videos HTML5.
</video>

<!-- Audio -->
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Tu navegador no soporta audio HTML5.
</audio>
