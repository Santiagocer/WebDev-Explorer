Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje que describe cómo se deben mostrar los elementos HTML en la pantalla.

¿Qué es CSS?

CSS es un lenguaje de hojas de estilo que describe la presentación de un documento escrito en HTML. CSS describe cómo deben mostrarse los elementos HTML en la pantalla, en papel, o en otros medios.

Sintaxis básica CSS

/* Selector de elemento */
p {
  color: blue;
  font-size: 16px;
}

/* Selector de clase */
.destacado {
  background-color: yellow;
  padding: 10px;
}

/* Selector de ID */
#principal {
  width: 80%;
  margin: 0 auto;
  color: red;
  font-weight: bold;
  text-align: center;
  font-size: 20px;
  border: 2px solid black;
  border-radius: 5px;
}

Flexbox Layout

Flexbox es un modelo de diseño unidimensional que permite distribuir el espacio entre los ítems de una interfaz.

Conceptos clave de Flexbox

Flexbox trabaja con un contenedor flex y sus hijos flex items. Las propiedades principales incluyen display: flex, flex-direction, justify-content, align-items y flex.

Código Flexbox

.contenedor {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
}

.item {
  flex: 1;
  padding: 1rem;
  background: #2965f1;
  color: white;
}

CSS Grid

CSS Grid Layout es un sistema de diseño bidimensional que permite crear layouts complejos con facilidad.

Grid vs Flexbox

Mientras Flexbox es unidimensional (trabaja en una fila o columna), Grid es bidimensional (trabaja con filas y columnas simultáneamente). Son complementarios y pueden usarse juntos.

Código Grid

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}

.grid-item {
  background: #2965f1;
  color: white;
  padding: 1rem;
  text-align: center;
}

.header {
  grid-column: 1 / 4;
}

.sidebar {
  grid-row: 2 / 4;
}

Animaciones y Transiciones

CSS permite crear animaciones fluidas sin necesidad de JavaScript.

Transiciones vs Animaciones

Las transiciones son cambios suaves entre estados, mientras que las animaciones permiten secuencias más complejas con múltiples keyframes.

Código de Animación

/* Transición */
.box {
  width: 100px;
  height: 100px;
  background: #2965f1;
  transition: all 0.3s ease;
}
.box:hover {
  transform: rotate(45deg);
  background: #6e48aa;
}

/* Animación */
@keyframes rebote {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}
.animado {
  animation: rebote 1s infinite;
}

Media Queries

Las media queries permiten aplicar estilos diferentes según características del dispositivo.

Diseño Responsivo

Las media queries son esenciales para crear diseños que se adapten a diferentes tamaños de pantalla. Los breakpoints comunes son 768px (tablets) y 480px (móviles).

Código Media Queries

/* Estilos base */
.container {
  width: 100%;
  padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    width: 980px;
  }
}
